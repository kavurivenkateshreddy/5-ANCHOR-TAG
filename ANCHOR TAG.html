ANCHOR TAG

--->The ANCHOR TAG (a> defines a hyper link which is used to link from one page to another page

ATTRIBUTE'S FOR ANCHOR TAG

  1. href---> hyper link referance Target Web page (ULR) where 'href' is the place holder for the hyper link

  2. Title---> The Title attribute specifices extra information about the element

    --->The informatin ismost often show as a 'tool tip' text when the mouse move over the element

  3. Target--->The Target attribute specifices where to open the linked document(in the same pageor different page)

APPERANCE OF HTML ANCHOR TAG

  1. --->An UNVISTED LINK is displayed under lined and blue

  2. ---> A VISITED LINK is displayed under lined and purpled

  3. ---> An ACTIVE LINK is underlined and red
